# Предметная область и требования к системе

### __Предметная область__

Есть автор (организация этого автора) или издательство, книги различных жанров, на которые у них есть право на распространение. 
Автор (авторы) лично или издательство (в лице уполномоченного человека) распространяют продукцию по точкам сбыта.
Покупатель приходит на точку сбыта и покупает книгу (книги). Я НЕ ЗНАЮ, ЧТО ТУТ ЕЩЕ ПИСАТЬ.

### __Требования к системе__

Система имеет базу электронных книг, предоставленных непосредственно автором или издательством. Книга представляет собой
саму книгу в электронном виде, различные параметры книги: название, авторы, жанры, издательство или отметка "самиздат", 
дата выпуска, статус книги, описание, цена, количество страниц, обложка, отзывы, возрастное ограничение, формат файла хранения, 
оглавление, серийный номер книги. Статус книги может иметь значения: "выпущена", "заблокирована", "скоро появится", "заявка". Цена 
устанавливается в соответствии с той ценой, которая согласована руководством сайта с распространителем. Описание - краткие, 
вводные сведения событий книги. Серийный номер книги - ISBN, номер присваиваемый книге при написании книги. Состоит из 10 цифр, 
если был присвоен до 2007 года, из 13, если был присвоен после 1 января 2007 года. Значение статусов книги: "скоро появится" - 
анонсирована, но еще не выпущена, "заявка" - книга отправлена на публикацию в системе.

У системы есть пользователи: авторизованные и неавторизованные. Неавторизованные пользователи имеют ограниченный функционал 
и никаким образом не хранятся в системе. Авторизованный/зарегистрированный пользователь имеет параметры: логин, пароль,
ФИ, e-mail, статус пользователя, статус аккаунта, Полномочие. Статус пользователя может иметь значения: "пользователь", 
"админ". Статус аккаунта определяется тем: подтвержден e-mail или нет, и может иметь значения: "подтвержден", "не подтвержден".
Полномочие указывает на то: имеет ли пользователь данного аккаунта право на взаимодействие с доп. функционалом, присущим
лицу уполномоченному издательством/личной организацией автора, и может иметь значения: "уполномочен", "неуполномочен". 
У уполномоченного лица указано: какой компанией данное лицо было уполномочено (организация-распространитель).

У авторизованного пользователя могут быть связи с книгами: книга может находиться у пользователя в корзине, личной библиотеке.

После совершения покупки в системе появляется запись о покупке с параметрами: позиции покупки, дата, сумма и тип. Тип может
принимать значения: "предзаказ", "покупка".

У каждой книги есть авторы. Они проставляются при заявке на публикацию книги в системе. Авторы либо выбираются из существующих
авторов в системе, либо заполняется поле "доп. авторы", которые не обязательны для отображения в системе как отдельных авторов, 
либо заполняется карточка нового автора. Новый автор будет добавлен в систему по принятии заявки на публикацию книги.
У автора есть параметры: ФИО, дата рождения/смерти, доп. информация, отзывы об авторе.

Каждую книгу выпускает либо издательство, либо компания самого автора. Организации-распространители имеют параметры: название, статус, 
автор, если компания - компания автора, отзывы на компанию, если она является издательством. Статус организации может иметь значения: 
"самиздат", "издательство".

У каждой книги есть жанры и теги. Они имеют название. 

Существуют отзывы на авторов, книги, компании, если они являются издательством. Эти отзывы являются разными сущностями,
но при этом имеют одинаковые параметры: текст отзыва, оценка, дата. Оценка - численная оценка от 1 до 5.

Каждый авторизованный пользователь, может добавлять авторов, компании, если они являются издательством, жанры и теги в
избранное.


Неавторизованный пользователь заходит на сайт. Может просмотреть список всех книг, жанров, авторов, издательств.
При поиске книг пользователь может производить поиск по различным параметрам (комбинациям параметров). Поиск автора, 
издания, жанров производится по наименованию. Пользователь может перейти на любую книгу, увидеть о ней подробную 
информацию, отзывы читателей с оценками, может перейти на автора книги, издательство, жанр. При просмотре авторов 
пользователь может перейти на любого из них, увидеть подробную информацию о нем, отзывы читателей с оценками, список самых 
популярных книг. При просмотре издательств пользователь может перейти на одно из, увидеть подробную информацию о нем,
список самых популярных книг. При просмотре жанров пользователь может перейти на один из них, увидеть список книг данного
жанра. Если пользователь хочет добавить книгу себе в корзину, он должен авторизоваться/зарегистрироваться в системе.

Авторизованный пользователь может выйти из аккаунта, изменить пароль. Может добавить/удалить книгу в свою корзину. Для оформления 
покупки пользователь обязан подтвердить свой e-mail. После подтверждения своего e-mail, пользователь может оплатить заказ. При 
успешной оплате книги переносятся из корзины в личную библиотеку. Из личной библиотеки пользователь может пользоваться 
купленными книгами, следить за книгой, купленной предзаказом. Также авторизованный пользователь может оставить отзыв
на купленную книгу, автора купленной книги, издательство купленной книги. Если книга, автор ил издательство нарушают 
какие-либо правила (моральные, законодательство и тд.), пользователь в праве отправить жалобу на нарушение. Также 
пользователи могут отмечать добавлять авторов и жанры в избранное.

Если пользователь является автором, он может подать заявку на установку статуса автора. Предоставляя необходимое 
подтверждение личности и подтверждение наличия произведений, на которые у него есть права, пользователь получает статус 
автора. 

Если пользователь является лицом представляющим какое-либо издательство, он может подать заявление на регистрацию 
издательства в системе. Регистрируясь пользователь обязан предоставить необходимые документы доказывающие, его
связь с издательством, а также документы подтверждающие существование данного издательства, должен указать,
пользователя в системе, который будет уполномочен управлять произведениями, выпущенными данным издательством.
После проверки доказательств, издательство регистрируется в системе, пользователь, указанный в заявке получает 
статус уполномоченного лица издательства.

Пользователи, являющиеся уполномоченным лицом какого-либо издательства или автором, могут подавать заявки на публикацию
произведений, предоставляя все необходимые подтверждения права на распространение данного произведения. Также в заявке
указываются такие параметры как: цена без наценки, название, автор, указание, если автор есть в системе, жанр, описание,
доступ к электронной версии книги и тд. После подтверждения книга публикуется в системе. Автор/лицо уполномоченное
издательством могут просматривать статистику по покупке своих произведений.

Администратор имеет возможность рассмотрения всевозможных заявок пользователей. После рассмотрения заявок, может 
отклонить или принять их (в том числе и жалобы), имеет доступ к расширенной информации аккаунтов пользователей. 
Имеет доступ к расширенной информации об авторах, издательствах, покупках, отзывах
